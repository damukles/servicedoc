# ServiceDoc

Document services and their dependencies and show them in a visual graph.

## Prepare dev env

```sh
npm install -g json-server
```

In the project folder:

```sh
elm package install
```


## Run dev env

```sh
./data/server.cmd
elm-app start
```

go to http://localhost:3000
